# Описание лабораторных и практических работ

## ПР1 — Дифференциальные уравнения и структурные схемы

1. **Получение структурной схемы по ДУ**  
   Выражается старшая производная, правая часть идёт на сумматор. Интеграторы понижают порядок до y.  
   Передаточная функция: члены с x → числитель, с y → знаменатель. Производные заменяются на степени s (оператор Лапласа).

2. **Получение ДУ из схемы**  
   Запись уравнений элементов, приведение к виду y слева, x справа.

3. **Схема с нелинейным элементом**  
   Аналогично примеру из задания. Используется функция Хевисайда: heaviside(t) = 0 при t < 0, 1 при t > 0.
